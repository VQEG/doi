# VQEG DOI Landing Page Builder

Static web app ready to be published with GitHub Pages.

## What it does

- Displays a form to enter publication metadata.
- Includes a dropdown with standard BibTeX entry types.
- Allows importing a BibTeX entry and automatically filling the form.
- Generates a live preview of the bibliographic landing page.
- Lets you copy a self-contained static HTML block.
- Lets you download a complete `.html` file ready to publish.

## Publishing on GitHub Pages

1. Push this content to the repository.
2. In GitHub, enable Pages for the main branch and the repository root.
3. The site will be served from `index.html`.

## Files

- `index.html`: main interface.
- `styles.css`: lightweight styling with a simple institutional look.
- `app.js`: form logic, BibTeX import, preview generation, and HTML export.
